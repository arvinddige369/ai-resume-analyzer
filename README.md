# AI Resume Analyzer

AI-powered tool that analyzes a resume against a job description and provides:

- Match score
- Matched skills
- Missing skills
- Suggestions to improve the resume

## Tech Stack

- Next.js
- React
- TypeScript
- OpenAI API
- Tailwind CSS

## Features

- AI resume analysis
- JSON structured output
- Rate limiting
- Input validation
- Secure API key handling

## Installation

```bash
npm install
npm run dev
